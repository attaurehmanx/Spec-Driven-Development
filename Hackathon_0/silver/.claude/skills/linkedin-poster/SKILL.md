# linkedin-poster

Helps create and manage LinkedIn posts for business promotion.

## Usage

```
/linkedin-poster
```

Then describe what you need:
- "Create a LinkedIn post about [topic]"
- "Draft a post for today"
- "Review pending LinkedIn drafts"

## What This Skill Does

1. **Post Creation**: Drafts LinkedIn posts based on your business and goals
2. **Content Guidelines**: Follows best practices for LinkedIn engagement
3. **Approval Workflow**: Creates drafts for human review before posting
4. **Scheduling**: Tracks posting frequency to maintain consistency
5. **Templates**: Uses proven post structures for maximum impact

## Post Guidelines

### Length
- 150-300 words optimal
- First 2 lines are critical (preview)

### Structure
1. **Hook**: Attention-grabbing opening
2. **Value**: Story, insight, or tip
3. **Call to Action**: What should readers do?
4. **Hashtags**: 3-5 relevant hashtags

### Topics
- Recent project successes
- Industry insights
- Problem-solving tips
- Behind-the-scenes
- Client testimonials
- Lessons learned

## Examples

### Create Post About Recent Project
```
/linkedin-poster
Create a LinkedIn post about successfully completing a client project. Focus on the problem we solved and the results achieved.
```

### Daily Post
```
/linkedin-poster
Draft today's LinkedIn post. Check Company_Handbook.md for current business focus.
```

### Review Drafts
```
/linkedin-poster
Review all pending LinkedIn drafts in LinkedIn_Drafts folder
```

## Workflow

1. Skill creates draft in LinkedIn_Drafts/ folder
2. Human reviews and edits draft
3. Human moves to Pending_Approval/ when ready
4. After approval, human posts manually to LinkedIn
5. Move to Done/ after posting

## Best Practices

- Post consistently (daily or 3-5x per week)
- Engage with comments within first hour
- Use storytelling, not just facts
- Include a clear call to action
- Tag relevant people/companies when appropriate
- Use line breaks for readability

## Hashtag Strategy

Include mix of:
- Broad industry tags (#Marketing, #Tech)
- Niche tags (#B2BSaaS, #ContentStrategy)
- Trending tags (check LinkedIn trends)
- Branded tags (your company hashtag)

Example: `#DigitalMarketing #ContentStrategy #B2B #MarketingTips #GrowthHacking`

## Posting Schedule

Optimal times (adjust for your audience):
- Tuesday-Thursday: 9-11 AM
- Avoid weekends for B2B
- Test different times and track engagement
